# Travel Agency Website – Portugal Guided Tours

A modern, responsive HTML/CSS template for travel agencies, tour operators, and local guides in Portugal.  
Designed to showcase destinations, tour packages, customer reviews, and a contact form.

## Features

- Fully responsive (works on desktop, tablet, mobile)
- Video backgrounds in hero and destination cards
- Tour plans with prices (editable menu blocks)
- Customer reviews timeline
- Google Maps integration
- Contact form (ready to connect to a backend)
- Clean, professional design based on Bootstrap 5

## Tech stack

- HTML5 / CSS3 / JavaScript
- Bootstrap 5
- Vegas Slider for background images
- jQuery (for some interactions)

## How to use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Customise content:
   - Replace images in `/images` folder.
   - Replace videos in `/videos` folder.
   - Edit tour prices, titles, and descriptions in `index.html`.
   - Update the contact email, phone, and address in the footer.
4. To add a backend (e.g., Flask, PHP), connect the reservation form and contact form to your own server.
